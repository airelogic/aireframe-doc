    Transfer Configuration

    Exporting and importing a tenants configuration

    Apr 11, 2023

    This page details how to export and import a configuration into Aire Frame. This allows you to easily transfer your configuration across multiple tenants.

    What can you find on this page:

    • Required Permissions
    • Export Configuration
    • Import Configuration

    Required Permissions

    • In order for you to export a configuration you will need the Export Configuration permission.
    • For you to import a configuration you will need the Import Configuration and then any of these following permissions depending upon what you are trying to import:
      • Create Structural Entities
      • Create Group
      • Create Data Sources
      • Edit Subject Configuration
      • Create Data Extract
      • Create Visualisation
      • Create Collection

    Export Configuration

    Exporting a configuration will generate a JSON output of the selected items within your current configuration ready to be imported into a different tenant. This json output can either be copied or downloaded onto your hard drive.

    Some items have particular rules to stop any conflicts and will therefore need to be exported together. Here is a list of the requirements necessary for each item:

    Dependencies Structure Groups Data Sources Subject Configuration Data Extracts Widgets Collections
    STRUCTURE   X          
    GROUPS X            
    DATA SOURCES       X X X X
    SUBJECT CONFIGURATION     X        
    DATA EXTRACTS     X     X X
    WIDGETS     X   X   X
    COLLECTIONS     X   X X  
    1. Go to the Transfer Configuration screen: Configuration -> Transfer
    2. Select which items you would like to export
    3. Click the export button
    4. You will get a JSON response as follows
    5. Either copy your json response into a text file or click the download button to download the JSON

    Import Configuration

    Importing a configuration will take a JSON or text file and upload it to the current configuration. This will update any items that it matches the key and type to and create any that it cannot find. This can be a risky endeavour so if any of the imports fail it will restore back to the latest version of the configuration.

    After clicking confirm you will be navigated away from the Transfer Configuration area while the job runs in the background. This may take some time however you will be notified after completion.

    1. Go to the Transfer Configuration screen: Configuration -> Transfer
    2. Select the Import tab from the top bar. You should see a screen as follows
    3. Either drag a file or click the draggable area and find the file which you would like to upload
    4. Click the Import button
    5. A warning will remind you that this could cause complications where you can either cancel or continue with the import
