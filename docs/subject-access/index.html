    Portal (Subject access)

    Configure self access for subjects

    Feb 16, 2024

    AireFrame provides a self-service portal for subjects to access their own data. The feature must first be enabled at a tenant level, then each user must be invited to access the portal.

    Required Permissions

    • In order to change the subject access configuration you will need the following permissions:
      • View Subject Configuration
      • Edit Subject Configuration
    • The following permissions are used to view and manage the subject access invites:
      • View Subject Access Invite
      • Create Subject Access Invite
      • Delete Subject Access Invite

    Enable Subject Access

    To enable the subject access portal, go to the subject configuration screen: Configuration -> Subject Configuration -> Subject Access.

    Linking Fields

    Linking fields are one or more unique fields that are used to link a user to a subject. It is the value that will be used to invite the subject user to access the portal. You must provide the claim type for the linking field, this is the field AireIdentity will use to invite the user.

    For example, if you choose the email address as the linking field with a claim type of email, then when you invite a subject with an email to access the portal, they will receive an email with a link to set up their account. Once they have set up their account, they will be able to see the data for the subject they were invited to access.

    Note, only the email claim type is currently suppported.

    Inviting a Subject

    To invite a subject, go to the subject you want to invite, click edit and then click the Invite button under the subject access section.

    Note, if there is no value for the linking field(s) then you will not be able to invite the subject.

    Managing Invites

    Once a user has been invited to access the portal, you can view and manage the invite on the edit subject screen.

    You can also see the user within the list of users where you will be able to see what subject they have access to.
