page-structural-entities ​

Specification Version ​

Version: "1"

Context and rules ​

Required for the SubjectRead capability.

Returns paged structural entities based on various criteria received by the endpoint.

Endpoint specification ​

  • Name: endpoint-prefix-aireframe-page-structural-entities
  • Verb: POST
  • Authentication: Client credentials
