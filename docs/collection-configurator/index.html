    Collection Configurator

    Configure Collection

    Apr 28, 2022

    This page explains how to create custom views in aireframe. For example you may want to show different groups of visualisations on the subject view and the multi subject view.

    What you can find on this page:

    • Required Permissions
    • Create Collection
    • Enabling Collections
    • Edit Collection
    • Delete Collection

    Required Permissions

    • In order to create collections you will the following permissions:
      • Create Collection
      • List Collection
      • View Collection
    • In order to edit a collection you will need the following permissions:
      • Edit Collection

    Create Collection

    1. Go to the subject configuration screen: Configuration -> Collections.
    2. Click add new. You will see the following screen
    3. Enter a title for your collection.
    4. Select the view you want this collection to be available on from the four options.
    5. Drag over the visualisations you want to enable from the Add Widgets section to the widgets input.
    6. Click save changes.

    Options: There are four customizable views in AireFrame:

    • Subject View - This is where you see the visualisation on the subject page with data only for that subject
    • Dashboard - This is where you see the visualisation on the structural entity overview page with data for all subjects located within that structural entity. Note that this is recursive - it shows data for subjects that are located in child structural entites and so on
    • Multi Subject View - This is where you see the data shown on the subject list alongside their id and name. On this view the visualisation is slightly altered to retrieve only the first data point for that subject (according to the default ordering)
    • Subject Preview - This is where you see a preview of the visualisations seen on the subject page with data only for that subject when you go to view the full subject data on a full screen.

    Enabling Collections

    Once you create a collection you can select it using the selector on each of the four customizable views

    .

    Edit Collection

    1. Go to the collection configuration screen: Configuration -> Collections.
    2. Select the collection you wish to edit.
    3. Enter the values you wish to change.
    4. Click save changes.

    Delete Collection

    1. Go to the collection configuration screen: Configuration -> Collections.
    2. Select the collection you wish to delete.
    3. Click remove dashboard.
